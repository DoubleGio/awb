  7. IPv6 (Internet Protocol version 6) also known as IPng (Internet Protocol next generation) is the newest version of the Internet Protocol. The main features of this protocol are providing identification and location system for computers on networks and routes traffic across the internet. The IPv4 (Internet Protocol version 4) is the fourth version of the IP used for identifying hosts on a network using an addressing system. The IPv4 makes use of a 32-bit address scheme. The total number of IP-addresses will be 2^32. The internet is growing so fast that eventually there will not be enough IP-addresses. One of the main reasons to move from IPv4 to IPv6 is that the newer version makes it possible to generate more IP-addresses using a 128-bit address scheme. Another reasons to move from IPv4 to IPv6 includes better end-to-end connectivity, faster routing and better security.
